# Endpoints


#### method: POST
#### path name: /organization/register
#### body params (inputs): 
```JSON
{
    "username": "username",
    "password": "password",
    "orgName": "orgName",
    "description": "description",
    "missionStatement": "missionStatement",
    "address": "address",
    "city": "city",
    "postalCode": "postalCode",
    "maxDistance": "maxDistance",
    "websiteLink": "websiteLink",
}
```
#### response: successful ? status(201) : status(406)
